## How to Capture Screenshots

### Google Sheets
1. Open a sheet with demo data
2. Run reconciliation via Clearledgr menu
3. Screenshot with sidebar visible

### Excel  
1. Sideload the add-in
2. Open with demo data
3. Screenshot with task pane visible

### Slack/Teams
1. Send a test notification
2. Screenshot the channel/chat

### Gmail/Outlook
1. Open an invoice email
2. Open the Clearledgr panel
3. Screenshot with panel visible

## Tips

- Use a clean browser/app window
- Hide personal information
- Use realistic but fake data
- Ensure Clearledgr branding is visible
- Crop to exact dimensions required
